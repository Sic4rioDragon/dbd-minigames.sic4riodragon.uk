# DBD Mini Games

Small Dead by Daylight inspired mini-games for dbd-minigames.sic4riodragon.uk.

This is not meant to directly clone DBDle. The idea is to take the daily guessing-game style and make my own version with extra modes, my own layout, and a DbD Guessr mode.

## Planned games

- DBD Guessr
  - Guess the realm/map from screenshots.
  - Currently under development.

- Killer Guess
  - Guess the killer from hints.

- Perk Guess
  - Guess perks from icons, text, or clues.

- Quote Guess
  - Guess the character from a quote.

- Lore Guess
  - Guess the character from lore text.

- Terror Radius Guess
  - Guess the killer from terror radius/audio clues.

- Emoji Guess
  - Guess killer/survivor from emoji hints.
